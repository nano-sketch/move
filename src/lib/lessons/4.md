# Dictionaries

Dictionaries are Python's most powerful data structure. They store data as key-value pairs, allowing you to quickly look up values using unique keys.

## What is a Dictionary?

A dictionary is like a real dictionary where you look up a word (key) to find its definition (value). In Python, keys can be strings, numbers, or tuples, and values can be anything.

```python
student = {
    "name": "Alice",
    "age": 20,
    "grade": "A"
}
```

## Creating Dictionaries

### Empty Dictionary

```python
empty_dict = {}
also_empty = dict()
```

### Dictionary with Data

```python
person = {
    "name": "Bob",
    "age": 25,
    "city": "New York"
}

numbers = {
    1: "one",
    2: "two",
    3: "three"
}
```

## Accessing Values

Use square brackets with the key:

```python
person = {"name": "Alice", "age": 30}
print(person["name"])
print(person["age"])
```

### Safe Access with get()

Use `get()` to avoid errors if key doesn't exist:

```python
person = {"name": "Alice"}
print(person.get("age"))
print(person.get("age", 0))
```

## Adding and Updating

### Add New Key-Value Pair

```python
person = {"name": "Alice"}
person["age"] = 25
person["city"] = "Boston"
```

### Update Existing Value

```python
person = {"name": "Alice", "age": 25}
person["age"] = 26
```

### Update Multiple Values

```python
person = {"name": "Alice", "age": 25}
person.update({"age": 26, "city": "Boston"})
```

## Removing Items

### Remove Specific Key

```python
person = {"name": "Alice", "age": 25, "city": "Boston"}
del person["city"]
```

### Remove and Return Value

```python
person = {"name": "Alice", "age": 25}
age = person.pop("age")
print(age)
```

### Remove Last Item

```python
person = {"name": "Alice", "age": 25}
item = person.popitem()
print(item)
```

### Clear All Items

```python
person = {"name": "Alice", "age": 25}
person.clear()
```

## Checking Keys

### Check if Key Exists

```python
person = {"name": "Alice", "age": 25}
if "name" in person:
    print("Name exists")

if "email" not in person:
    print("Email doesn't exist")
```

## Dictionary Methods

### Get All Keys

```python
person = {"name": "Alice", "age": 25, "city": "Boston"}
keys = person.keys()
print(keys)
```

### Get All Values

```python
values = person.values()
print(values)
```

### Get All Key-Value Pairs

```python
items = person.items()
print(items)
```

## Looping Through Dictionaries

### Loop Through Keys

```python
person = {"name": "Alice", "age": 25}
for key in person:
    print(key)
```

### Loop Through Values

```python
for value in person.values():
    print(value)
```

### Loop Through Key-Value Pairs

```python
for key, value in person.items():
    print(f"{key}: {value}")
```

## Nested Dictionaries

Dictionaries can contain other dictionaries:

```python
users = {
    "user1": {
        "name": "Alice",
        "age": 25
    },
    "user2": {
        "name": "Bob",
        "age": 30
    }
}

print(users["user1"]["name"])
```

## Dictionary Comprehension

Create dictionaries using comprehension:

```python
squares = {x: x**2 for x in range(5)}
print(squares)

names = ["Alice", "Bob", "Charlie"]
name_lengths = {name: len(name) for name in names}
```

## Copying Dictionaries

### Shallow Copy

```python
original = {"name": "Alice", "age": 25}
copy1 = original.copy()
copy2 = dict(original)
```

## Common Use Cases

### Counting Items

```python
text = "hello world"
char_count = {}
for char in text:
    if char in char_count:
        char_count[char] += 1
    else:
        char_count[char] = 1
```

### Grouping Data

```python
students = [
    {"name": "Alice", "grade": "A"},
    {"name": "Bob", "grade": "B"},
    {"name": "Charlie", "grade": "A"}
]

by_grade = {}
for student in students:
    grade = student["grade"]
    if grade not in by_grade:
        by_grade[grade] = []
    by_grade[grade].append(student["name"])
```

## Common Mistakes

**Accessing non-existent key:**
```python
person = {"name": "Alice"}
print(person["age"])
```

**Using mutable keys:**
```python
my_dict = {[1, 2]: "value"}
```

**Modifying during iteration:**
```python
data = {"a": 1, "b": 2, "c": 3}
for key in data:
    del data[key]
```

## Key Points

- Dictionaries store key-value pairs
- Keys must be unique and immutable
- Access values using square brackets or `get()`
- Use `in` to check if key exists
- Methods: `keys()`, `values()`, `items()`
- Can loop through keys, values, or both
- Dictionaries are mutable and unordered
- Use dictionaries for fast lookups by key

Dictionaries are incredibly versatile and one of the most commonly used data structures in Python programming.
