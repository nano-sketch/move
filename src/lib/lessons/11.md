# String Methods & Special Characters

Python provides powerful string methods and special characters for text manipulation. Mastering these tools is essential for processing and formatting text data.

## Case Conversion Methods

```python
text = "Hello World"
print(text.upper())
print(text.lower())
print(text.capitalize())
print(text.title())
print(text.swapcase())
```

## Whitespace Methods

```python
text = "  hello  "
print(text.strip())
print(text.lstrip())
print(text.rstrip())
print("...hello...".strip("."))
```

## Search Methods

```python
text = "Hello World"
print(text.find("World"))
print(text.count("l"))
print(text.startswith("Hello"))
print(text.endswith("World"))
```

## Replacement Methods

```python
text = "Hello World"
print(text.replace("World", "Python"))
print("one one one".replace("one", "two", 2))
```

## Split and Join

```python
text = "apple,banana,orange"
fruits = text.split(",")
print(fruits)

words = ["Hello", "World"]
text = " ".join(words)
print(text)
```

## Alignment Methods

```python
text = "Hello"
print(text.center(20, "*"))
print(text.ljust(20, "-"))
print(text.rjust(20, "-"))
print("42".zfill(5))
```

## Special Characters

```python
print("Line 1\nLine 2")
print("Name:\tJohn")
path = "C:\\Users\\Documents"
print(path)
text = "He said, \"Hello!\""
print(text)
```

## Raw Strings and Unicode

```python
path = r"C:\Users\Documents"
print(path)
print("\u0041")
print("\N{GREEK SMALL LETTER ALPHA}")
```

## Formatting Methods

```python
name = "Alice"
age = 25
text = f"My name is {name} and I am {age} years old"
print(text)
number = 3.14159
print(f"{number:.2f}")
print(f"{42:05d}")
```

## Advanced Methods

```python
text = "Hello"
encoded = text.encode("utf-8")
print(encoded)

text = "Hello-World-Python"
parts = text.partition("-")
print(parts)
```

## Practical Examples

```python
def clean_input(text):
    return " ".join(text.strip().lower().split())

print(clean_input("  HELLO   WORLD  "))

text = "Hello"
print(text[::-1])

def is_palindrome(text):
    text = text.lower().replace(" ", "")
    return text == text[::-1]

print(is_palindrome("racecar"))
```

## String Slicing

```python
text = "Hello World"
print(text[0:5])
print(text[:5])
print(text[::2])
print(text[::-1])
```

## Common Patterns

```python
csv_line = "John,Doe,25,New York"
fields = csv_line.split(",")
first_name, last_name, age, city = fields
print(f"Name: {first_name} {last_name}")

url = "https://example.com" + "/api/users" + "?id=123"
print(url)
```

## Key Points

- Strings are immutable in Python
- Use `upper()`, `lower()`, `title()` for case conversion
- `strip()`, `lstrip()`, `rstrip()` remove whitespace
- `split()` and `join()` for string/list conversion
- `replace()` substitutes substrings
- Special characters: `\n` (newline), `\t` (tab), `\\` (backslash)
- Use raw strings `r""` for paths and regex
- F-strings are the modern way to format strings
- String methods return new strings, don't modify original
- Use `find()` for safe searching, `index()` raises error

String manipulation is fundamental to almost every Python program. These methods will be used constantly in real-world applications.
